How to play a text adventure game

Text adventures (sometimes also called "interactive fiction") describe the game world using text descriptions. Each location displays a its description, a list of objects that your character can interact with, and a list of exits. These games are usually controlled by typed-in commands addressed to the playable character. While playing text adventures, especially the ones from the 1980s, one would usually take notes and draw a map on paper. In this particular game, you type in commands and confirm them with the Enter key, similarly to the use of command line in older operating systems. The game understands simple commands consisting of a verb and an object, such as:

EXAMINE TAIL
TAKE AXE

You may but do not have to type articles. You can shorten commands to the first three letters, using, for example, the following versions of the commands:

EXA TAI
TAK AXE

To move between locations, type in the direction you want to go. You can shorten the directions to just the first letter, such as:

LEFT (or L)
RIGHT (or R)

The exception is the direction INSIDE, which cannot be shortened to I, because this command is reserved for the INVENTORY.

Most puzzles within the game can be solved by using the right object in the right location. You can only manipulate with objects that you can see or that you are carrying. You can carry up to two things at once, making it necessary to drop items that you are not going to need at the moment. You can list the items you are carrying by typing "INVENTORY" or "I".

You can list all the verbs used by the game by typing "DICTIONARY" or "VERBS". Please note that - as in the original - the dictionary of this game is quite limited, and you will have to use the generic verb USE to execute a wide range of actions. If you are stuck, the "HINT" command will display a hint for the current location. This hint was, however, not featured in the original game.

You can save and load your game while playing, using the SAVE and LOAD commands in conjunction with a label, for example:

SAVE HORSE
LOAD HORSE

If you don't use a label, the game will automatically call you savegame "SAVE". You can list all savegame labels by typing "GAMES". The savegames are being stored in the local storage of your web browser. As such, they will persist even when you close the game window but will be deleted when the local storage is cleaned by the browser.

O hře Indiana Jones na Václavském náměstí

Díky neoficiálním textovkám Františka Fuky a dalších autorů se Indiana Jones stal jedním z nejpopulárnějších hrdinů československých her. V tomto anonymně vydaném titulu z roku 1989 se americký archeolog překvapivě ocitá na Václavském náměstí v Praze během Palachova týdne – série protirežimních demonstrací, jež byly brutálně potlačeny příslušníky Veřejné bezpečnosti a Lidových milicí. Indiana Jones se musí probít skrz náměstí a odjet na letiště, což není vůbec snadné, protože na každém kroku čekají krvežízniví policisté a milicionáři. Tato mimořádně násilná hra umožňuje hráči prožít trauma Palachova týdne z pohledu akčního hrdiny, který se na rozdíl od skutečných demonstrantů mohl bránit. Autora či autorku skrývající se pod pseudonymem "Zuzan Znovuzrozený" neznáme, ale mimořádně vyhrocený tón i geograficky věrné zpracování napovídají, že se možná demonstrace zúčastnil/a.

Jak jsme hru rekonstruovali

V emulátoru ZX Spectra jsme analyzovali lokace a hádanky a z dochovaného binárního kódu hry jsme extrahovali veškeré texty. Poté jsme hru přepsali pro webové technologie (JavaScript, HTML a CSS) za použití vlastního na míru naprogramovaného enginu. Texty jsou původní, pouze s přidanou diakritikou a případnou opravou překlepů. Hádanky a jednotlivé herní mechaniky jsme se snažili co nejvěrněji reprodukovat. Oproti původní verzi jsou však v naší konverzi i následující změny:

  • Úvodní obrázek byl nakreslen pro potřeby remaku v rozlišení a paletě odpovídajícím grafickým možnostem původní platformy (ZX Spectrum). Původní verze žádný obrázek neměla.
  • Konverze umožňuje použití některých synonym (VEZMI místo SEBER apod.)
  • Konverze obsahuje nápovědy (přes příkaz POMOC), které nebyly v původní hře přítomny.
  • Konverze umožňuje uložit a nahrát pozici ve hře (SAVE a LOAD).
  • Konverze používá čitelnější písmo a má přívětivější a rychlejší uživatelské rozhraní. Umožňuje například opakovat předchozí příkazy pomocí kurzorových šipek, případně klávesy TAB.
  • Interface remaku umožňuje opakovat příkazy pomocí šipek nahoru a dolů, případně klávesy TAB.
  • Z konverze jsme vypustili úvodní instrukce, které vysvětlují, jak zadávat příkazy. Instrukce jsou nahrazeny touto nápovědou a funkcí SLOVNÍK.

Často kladené otázky (FAQ)

Zasekl/a jsem se a nevím, jak dál. Co mám dělat?
Zkuste použít příkaz POMOC. V případě největší nouze najdete kompletní návod na konci této stránky.

Hra nerozumí mým příkazům. Co s tím?
Tento titul má poměrně omezený slovník. Namísto specifických sloves jako "hoď" nebo "sekej" napište příkaz "POUŽIJ". Názvy předmětů pište od začátku. Tj. místo "MRTVOLU POLICAJTA" můžete napsat "MRTVOLU" nebo "MRT", ale nikoli "POLICAJTA".

Proč nedostanu čas na to zneškodnit nepřátele, i když vím jak?
Zkuste je zneškodnit z jiné lokace. Použití některých předmětů může mít dopad na jiném místě.

Je možné dostat se do stavu, kdy už hru nejde dohrát?
Ano, je to možné. V takovém případě se objeví zpráva, která vám to sdělí. Potom můžete nahrát pozici nebo hru spustit znovu pomocí příkazů LOAD a RESTART.
